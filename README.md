## ğŸš€ How It Works (High-Level)

1. **`run.py`**

   * Starts a background updater thread
   * Launches the Flask server
   * Waits for server readiness
   * Automatically opens the dashboard in your browser

2. **Background Updater**

   * Periodically refreshes TSE options data
   * Saves processed results into Excel files

3. **Flask Server (`server/app.py`)**

   * Serves the dashboard UI
   * Exposes REST APIs for:

     * Market status
     * Full option chain
     * Implied volatility history

4. **Frontend (HTML/CSS)**

   * Fetches data via APIs
   * Displays market metrics interactively
